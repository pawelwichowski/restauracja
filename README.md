# Restauracja

Responsywna lista restauracji z ocenami i filtrowaniem.

## Uruchomienie

Wymagany jest Node.js 20.19+ lub 22.12+.

1. npm install
2. npm run dev

Etap 1 używa React, Vite, JavaScript i CSS Flexbox. Dane demonstracyjne są w src/data/restaurants.js.

## Funkcje

- lista restauracji z ocenami,
- filtrowanie po nazwie, kuchni i minimalnej ocenie,
- sortowanie,
- responsywny układ.

Kolejny etap: backend Django z MySQL i API.
